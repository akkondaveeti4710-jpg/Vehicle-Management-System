# Vehicle Management System

## Description
This is a simple Python program that helps manage vehicle parking information. The program asks the user for the vehicle number, parking duration, and entry time. When the vehicle leaves, the program calculates the parking fee.

The information is saved in a text file so it can be stored for records.

## Features
- Enter vehicle number
- Enter parking duration
- Enter entry time
- Calculate parking fee
- Save vehicle information to a text file

## Parking Fees
30 minutes = $5  
1 hour = $10  
1 hour and 30 minutes = $15  
2 hours = $20  

If the parking time goes over the maximum time, an extra $10 fee may be added.

## How to Run
1. Make sure Python is installed.
2. Run the Python file.
3. Follow the instructions in the program.

## File Used
The program saves information in a file called:

vehicle_parking.txt
